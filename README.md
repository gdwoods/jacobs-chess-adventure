## How to Play

Simply open `index.html` in any web browser - no installation needed!

```bash
open index.html
```

### Or double-click the `index.html` file in Finder.

## Features

- **Kid-Friendly Design**: Colorful, animated interface that's fun to use
- **Personalized Messages**: Encouraging messages throughout the game just for Jacob
- **Learn-as-you-Play**: Visual highlights show where pieces can move
- **Helpful Hints**: Click the hint button when stuck
- **Undo Moves**: Made a mistake? No problem - undo and try again!
- **Friendly AI Opponent**: A chess buddy that's challenging but beatable
- **Move History**: See all the moves made during the game
- **Celebrations**: Special animations when Jacob wins!

## Game Controls

- **Click a Piece**: Shows all valid moves highlighted on the board
- **Click a Highlighted Square**: Moves the piece there
- **New Game**: Start a fresh game
- **Undo Move**: Take back your last move
- **Give me a Hint**: Get a suggested move

## Chess Tips for Beginners

1. **Control the center** - The middle of the board is important!
2. **Develop your pieces** - Move your knights and bishops early
3. **Protect your King** - Castle when you can
4. **Look for captures** - Free pieces are always good!
5. **Think ahead** - What will happen after your move?
